# kashira2339's TypeScript & Next.js

## How to use

### Instartion

```bash
git clone --depth=1 --branch=master https://github.com/kashira2339/next-typescript-boilerplate.git my-app
cd my-app
rm -rf !$/.git
npm install
```

### Development

```bash
npm run dev
```

### Production

```bash
npm run build
npm run start
```
